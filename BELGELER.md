- Visual Studio 2022 veya VS Code

```bash

git clone https://github.com/tda45/Chatter-LiveChat-PC.git
cd Chatter-LiveChat-PC

# Bağımlılıkları yükle
dotnet restore

# Çalıştır
dotnet run --project ChatterLiveChat
